intel Core i5 13400F
intel Core i9 13900KF

蓝色的表示代数，相当于制程工艺更新换代的级别，越大则技术越新。

红色的表示数值定位，基本上数字越大，CPU性能越强。
K是可超频，F是无核显
超频会影响CPU寿命，不建议使用。

注意：
intel Core i7 870
为第一代处理器，实际相当于
intel Core i7 1870

R5 1400
R3 2200G
R5 2600X
R7 3700X
R7 3900XT
R5 5600X
R7 7800X3D

2017年后，AMD锐龙处理器对标英特尔酷睿，在这之前都被intel吊着锤
命名方式类似

G是带核显版本 X是加强版 XT表示加强版又改良加强版
X3D表示加强版又提高CPU三级缓存，在一些很吃配置的游戏里可以提升一些性能
